# RV32IM-5Stage-Pipelined-Processor-with-AI-Extensions
Synthesizable 5-Stage Pipelined 32-bit RV32IM RISC-V Processor with Custom AI Instructions (VDOT4, VMAX4). Implements hazard detection, data forwarding, and EX-stage branch resolution. Verified in QuestaSim and synthesized using Synopsys Design Compiler (SAED32nm, 60ns constraint, WNS +54.22ns). Includes RTL, testbench, and reports.
